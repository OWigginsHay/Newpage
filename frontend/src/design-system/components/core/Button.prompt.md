Newpage's primary call-to-action — a bold, pill-shaped button. Use `primary` (solid teal) for the main action, `secondary` (teal outline) for supporting actions, `dark` on light imagery, and `ghost` for low-emphasis inline actions.

```jsx
<Button variant="primary" arrow>Explore AI Solutions</Button>
<Button variant="secondary">Learn more</Button>
```

Variants: `primary` | `secondary` | `dark` | `ghost`. Sizes: `sm` | `md` | `lg`. Set `arrow` for the up-right arrow, `as="a"` + `href` for links.
